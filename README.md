# Echo Map - 第一次旅游的贴心助手

Echo Map 是一款专为“第一次去某个地方”设计的旅游小程序。它旨在解决初次到达陌生城市时的迷茫，通过地图交互，即时为你提供最地道的省份识别与精选旅游方案。

## 🌟 核心理念
**“初见，即深知。”**
很多人在第一次到达一个新省份或城市时，不知道该看什么、该如何规划。Echo Map 结合地理位置信息，为你推荐那些“第一次去必看”的经典与地道体验。

## ✨ 主要功能
- **智能位置感知**：自动识别当前经纬度，并离线解析所在省份。
- **地图探索模式**：点击地图上的任意位置，立即获取该地区的专属旅游推荐。
- **精选视觉方案**：结合高质量图片展示，让你在出发前就能感受到目的地的风情。
- **极简交互设计**：底部弹窗式设计，信息展示与地图探索无缝衔接。

## 🛠 技术实现
- **前端**：原生微信小程序开发。
- **后端**：微信云开发（Cloud Development）+ 自建后端接口支持。
- **地理信息**：内置高效的离线地理位置逆编码逻辑。

## 📂 项目结构
- `pages/index`：主地图交互页面。
- `utils/geo.js`：核心地理位置解析算法。
- `assets/raw_images`：高质量原始素材存放区（不参与编译）。
